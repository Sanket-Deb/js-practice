# JS Interview Practice

A collection of JavaScript questions with solutions.

Each question has:

- `.js` file → solution
- `.md` file → question description

## Scripts

- `npm run new -- "Question Title"` → creates boilerplate files for new question
- `node questions/Q1_placeholder.js` → run a specific question file

## Question List

1. to find the first non repeating character in a string.
2. reverse the sequence of words in a given string.
